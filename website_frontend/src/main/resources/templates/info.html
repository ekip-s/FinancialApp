finansicli

Описание приложения.

Приложение для учета финансов. Сюда можно вносить ежемесячные траты, кредиты и прибыль в виде зарплаты. Подключать к своему финансовому состоянию родственников. Приложение в основном для обучения меня разным техноллогиям и применение их на практике.

Здесь указаны какие технологии использованы в приложении:

Spring Framework Нужно будет обновить версию spring-boot-starter-parent после обновления: Spring Security. Так как websecurityconfigureradapter deprecated.
PostgreSQL v15.2
Docker Сейчас приложение в 2-ух контейнерах: БД и само приложение. Далее разделим как минимум фронт с бэком. Еще скорее всего сделаю отдельный сервис статистики с конектом через Apache Kafka. Далее можно подключить телеграм бота, тоже в отдельной коробке(получится 2 фронта в отдельных коробках).
Nginx Здесь пока мало настроек, в основном редиректы. Далее можно улучшить.
Java Сейчас 11-ая. После обновления Spring Security, нужно будет перейти на 17-ую.
